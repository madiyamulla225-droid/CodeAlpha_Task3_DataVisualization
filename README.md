# CodeAlpha_Task3_DataVisualization
A data visualization project using Power BI to transform the Sample Superstore dataset into interactive charts, graphs, and dashboards that provide meaningful business insights and support data-driven decision-making.

## Project Title
Data Visualization using Power BI

## Objective
The objective of this project is to transform raw data into meaningful visualizations that help identify trends, patterns, and business insights for better decision-making.

## Dataset
Sample - Superstore Dataset

## Tool Used
- Microsoft Power BI

## Visualizations Created
- KPI Cards (Sales, Profit, Orders)
- Monthly Sales Trend
- Sales by Region
- Sales by Category
- Profit by Category
- Sales by Segment
- Sales by Ship Mode
- Interactive Slicers

## Key Insights
- Technology category generated the highest sales.
- The West region achieved the highest revenue.
- Standard Class was the most frequently used shipping mode.
- Monthly sales showed fluctuations throughout the year.
- Interactive dashboards help users explore data efficiently.

## Files Included
- Power BI Dashboard (.pbix)
- Sample Superstore Dataset (.xlsx/.csv)
- Project Report (.pdf/.docx)

## Conclusion
This project demonstrates how Power BI can convert raw data into interactive dashboards that support business analysis and data-driven decision-making.
